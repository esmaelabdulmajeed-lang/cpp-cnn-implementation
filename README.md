# CNN from Scratch in C++

This project is a C++ implementation of a Convolutional Neural Network (CNN)
built from scratch for academic purposes.

## Files
- `.h` header files (in include/)
- `.cpp` source files (in src/)
- `Makefile` to build the project
- `batman.ppm` sample input image (in data/)
